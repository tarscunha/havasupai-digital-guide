# **App Name**: Havasupai Tribe Digital Guide

## Core Features:

- Information Hub: Displays essential visitor information, reservations details, and cultural insights of the Havasupai Tribe.
- Intuitive Navigation: Allows users to quickly navigate to different sections such as reservations, visitor info, and contact details with a sticky header and smooth scrolling.
- Visual Showcase: Presents a gallery of high-quality images and maps showcasing the beauty and layout of the Havasupai Reservation and surrounding trails.
- Language Guide: AI tool providing real-time translations for key phrases in the Havasupai language to promote cultural understanding for visitors.
- AI Packing Assistant: Generates packing checklists based on user travel dates and reservation type (lodge vs. camping) to prepare for the trip. Tool makes choices about gear suitability.
- Vendor Directory: Displays a list of approved vendors for hiking equipment, local guides and shuttle options.

## Style Guidelines:

- Primary color: #20B2AA (Light Sea Green) evoking the blue-green waters of Havasupai.
- Background color: #E0FFFF (light cyan) providing a clean and calming backdrop.
- Accent color: #B22222 (Firebrick) draws attention to calls to action, and contrasts with the earth tones of the canyon.
- Font pairing: 'Space Grotesk' (sans-serif) for headers, giving a modern, clean feel combined with 'Inter' (sans-serif) for body text.
- Use of clear, simple icons for navigation and information points to enhance user experience.
- Clean, card-based layouts for content sections to improve readability and overall aesthetic appeal.
- Subtle transitions and hover effects on interactive elements for user feedback.
